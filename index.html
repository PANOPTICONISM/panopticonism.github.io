MY AREAS OF EXPERTISE

Visual Identity

Every brand needs a strong visual identity to be recognised and to grasp the target's audience attention. By creating a consistent visual across media platforms, goods and the brand as a whole, I can assist in that mission.

Examples include designs of, goods, logo, website and graphics.

UI/UX Design

Research and design are like opposite sides of the same coin. My designs are based on factual research done beforehand to achieve the utmost efficience while proving the target audience with the best possible user experience.

Examples of research include, BERT test, usability test, card sorting and desktop research.

Development

As a web developer, I hold the range to develop a simple static page or a more intricate website with dynamic data with WordPress for backend.

In simpler words, I can build a page or more with content given by you, the client, or a whole web shop from which you'll be able to edit the content with new products.
